# 🌱 MISIÓN: ADAPTACIÓN

### El juego interactivo que enseña que el duelo no se olvida, se integra

Juego de mesa digital psicoeducativo para estudiantes, docentes y familias.
Duración: 15-20 minutos · 1 a 4 jugadores

**Jugar online:** https://TU-USUARIO.github.io/mision-adaptacion/

---

### ¿Cómo jugar?

1. Elige cuántos jugadores/equipos participan y su Espíritu Compañero
2. Lanza el dado y avanza por el tablero
3. Responde tarjetas de 3 colores:
   - 🟩 **VERDE - Adaptación Activa**: Cómo acompañar el duelo
   - 🟨 **AMARILLO - Mito o Realidad**: Verdadero / Falso sobre el duelo
   - 🟦 **AZUL - Integración Emocional**: ¿Estrategia válida o evasión?
4. Para ganar en **NUEVO EQUILIBRIO** debes haber respondido al menos 1 tarjeta de cada color

> "No olvidé, aprendí a vivir con ello."

---

### Casillas especiales
- 🪤 **La Cueva del Olvido**: Pierdes un turno. Olvidar no es la solución.
- 🌉 **El Puente de la Validación**: Avanzas 3 casillas. Validar el dolor acelera la adaptación.

---

Hecho con React + TypeScript + Tailwind CSS
100% client-side, sin base de datos, funciona offline.
